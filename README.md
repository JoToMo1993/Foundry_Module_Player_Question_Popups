# Foundry Player Question Popups

This module allows you to easily send question to players and receive secret responses back.
In combination with a macro you can easily create questions that will be send to your players and their choices will be
whispered to you via the game-chat.
This way you can reveal the results at the end, if you want to.

## Setup / Installation

To install this module, you have to add the module.json's address into the module-manager of foundry.
The url
is https://raw.githubusercontent.com/JoToMo1993/Foundry_Module_Player_Question_Popups/refs/heads/main/module.json
